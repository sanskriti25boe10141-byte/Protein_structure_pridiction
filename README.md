# Protein_structure_pridiction
Protein Structure Prediction of Human Myoglobin (P02144) using AlphaFold DB and Chimera, including 3D structure visualization, analysis, and scientific documentation.
# Task 3: Protein Structure Prediction

## Objective
To predict and visualize the three-dimensional (3D) structure of a protein using bioinformatics tools.

## Protein Selected
- Protein Name: Myoglobin
- UniProt ID: P02144
- Organism: Homo sapiens

## Tools Used
- UniProt
- AlphaFold DB
- ChimeraX

## Methodology
1. Selected the Myoglobin protein sequence from UniProt.
2. Retrieved the predicted 3D structure using AlphaFold DB.
3. Visualized the protein structure in ChimeraX.
4. Captured structure images for analysis.
5. Examined structural features and confidence scores.

## Structural Features
- Myoglobin is an oxygen-binding protein found in muscle tissues.
- The structure is predominantly composed of alpha-helices.
- AlphaFold prediction showed very high confidence for most residues.
- The protein exhibits a compact globular fold.

## Results
The predicted 3D structure was successfully obtained and visualized. Structure images were generated and analyzed to understand the protein's organization and function.

## Conclusion
The 3D structure of Human Myoglobin (P02144) was predicted using AlphaFold DB and visualized using ChimeraX/PyMOL. The study demonstrates the use of computational tools for protein structure analysis.
